## How to use

1. Download the project
```bash
$ git clone https://gitbub.com/cadenzah/pos.git
```
2. Locate into project's folder
3. Install npm modules
```bash
$ cd pos
$ npm install
```
4. Build React source
```bash
$ npm run build
```
5. If the build process was successful, Start the service
```bash
$ npm run start
```
6. Use the service
```bash
http://localhost:3000
```
